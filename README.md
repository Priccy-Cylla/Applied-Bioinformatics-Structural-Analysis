# Structural-Bioinformatics-and-Sequence-Analysis

This is an assignment for the Structural Bioinformatics Internship program. This repository entails biological data retrieval from three major databases, that is NCBI, Protein Data Bank (PDB), and UniProt.

# Overview

This project is a practical series and a hands-on implementation of computational structural biology concepts. It focuses on sequence retrieval and analysis utilising bioinformatics tools and databases.

# Objectives

Retrieve biological data from 3 major databases:
1. NCBI
2. Protein Data Bank (PDB)
3. UniProt
